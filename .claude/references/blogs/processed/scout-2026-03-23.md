## Official Anthropic skills repo analysis

The official `anthropics/skills` repo (101k stars) and the Agent Skills spec at agentskills.io define the canonical skill format.

| Field/Convention           | Official spec                                                        | Our framework                                        | Gap                                                         |
| -------------------------- | -------------------------------------------------------------------- | ---------------------------------------------------- | ----------------------------------------------------------- |
| `name` (required)          | Max 64 chars, lowercase+hyphens, must match directory name           | ✅ All skills use lowercase names matching directory | None                                                        |
| `description` (required)   | Max 1024 chars, describe what + when to use                          | ✅ All skills have descriptions                      | None                                                        |
| `license` (optional)       | License name or reference to bundled file                            | ❌ Not used                                          | Low priority — add if publishing as open-source plugin      |
| `compatibility` (optional) | Max 500 chars, environment requirements                              | ❌ Not used                                          | Medium — could specify "Requires Claude Code v2.1.71+"      |
| `metadata` (optional)      | Arbitrary key-value for additional properties                        | ❌ Not used                                          | Low — could add `author`, `version` for plugin distribution |
| `allowed-tools` (optional) | Space-delimited list of pre-approved tools                           | ✅ Used in `/2_review`                               | Partially — could add to more skills for safety             |
| `disable-model-invocation` | Claude Code extension (not in Agent Skills spec)                     | ✅ Used on all user-invoked skills                   | N/A — CC-specific field                                     |
| `effort`                   | Claude Code extension                                                | ✅ Added to key skills                               | N/A — CC-specific field                                     |
| `argument-hint`            | Claude Code extension                                                | ✅ Used on skills with arguments                     | N/A — CC-specific field                                     |
| `model`                    | Claude Code extension                                                | ✅ Used on key skills                                | N/A — CC-specific field                                     |
| `scripts/` directory       | Executable code agents can run                                       | ❌ Not used                                          | Medium — `/smoke`, `/create-hook` could bundle scripts      |
| `references/` directory    | Additional docs loaded on demand                                     | ❌ Skills don't use per-skill references             | Medium — could move skill-specific docs from `context/`     |
| `assets/` directory        | Templates, images, data files                                        | ❌ Not used                                          | Low — `/create-hook` could bundle hook templates            |
| Progressive disclosure     | Metadata ~100 tokens, instructions <5000 tokens, resources on demand | ⚠️ Some skills exceed 5000 token recommendation      | Review skill sizes and split large ones                     |
| Name must match directory  | `name` field = parent directory name                                 | ✅ Already matches                                   | None                                                        |
